# clay-date-picker

ClayDatePicker

## Setup

1. Install NodeJS >= v6.0.0 and NPM >= v3.0.0, if you don't have it yet. You
can find it [here](https://nodejs.org).

2. Install local dependencies:

  ```
  yarn install
  ```

3. Start the code:

  ```
  yarn run start
  ```
